## Arquitectura del proyecto

Carpeta raíz:

- `Index.html` - Página principal.
- `Ofertas.html` - Página de ofertas.
- `README.md` - Documentación del proyecto.

Carpeta `src/`:

- `src/pages/Index.html` - Página principal.
- `src/pages/Ofertas.html` - Página de ofertas.
- `src/css/Index.css` - Estilos para la página principal.
- `src/css/Ofertas.css` - Estilos para la página de ofertas.
- `src/js/Index.js` - Interactividad de la página principal.
- `src/js/Ofertas.js` - Interactividad de la página de ofertas.
- `src/img/` - Imágenes usadas por el proyecto.

## Cómo usar

Abre `Index.html` en el navegador para ver la página principal.

La estructura separa claramente:

- contenido (`.html`)
- estilos (`src/css/`)
- scripts (`src/js/`)
- recursos (`src/img/`)
